# MobWeb_UseShortDescriptionAsMetaDescriptionForProduct extension for Magento

If no meta description attribute is set for a product, use the product's short description as the meta description on the product page in the frontend.

## Installation

Install using [colinmollenhour/modman](https://github.com/colinmollenhour/modman/).
